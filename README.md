# Smart Warehouse Management System

A Java-based warehouse inventory and shipment management system developed for a Data Structures course project.

## Technologies

- Java 17
- Maven
- PostgreSQL
- Docker Compose

## Main Data Structures

- Hash Table
- Queue
- Priority Queue / Heap
- Binary Search Tree
- Linked List
- Stack
- Graph

## Algorithms

- Binary Search
- Merge Sort
- Dijkstra Algorithm
